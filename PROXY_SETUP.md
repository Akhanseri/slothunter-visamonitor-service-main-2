# Настройка прокси для обхода блокировки IP

## Проблема

IP сервера (89.207.255.163) заблокирован на стороне visa API. Все запросы возвращают "Connection refused" на порту 443.

## Решение

Использовать HTTP/HTTPS прокси для обхода блокировки.

## Настройка

1. Получите прокси (HTTP/HTTPS):
   - Платные прокси-сервисы (Bright Data, Smartproxy, Oxylabs и т.д.)
   - Бесплатные прокси (менее надежные)
   - Собственный прокси-сервер

2. Добавьте `PROXY_URL` в `.env.production`:

```bash
# Формат: http://username:password@proxy-host:port
# Если пароль содержит спецсимволы (+, @, #, % и т.д.) — закодируйте их:
#   + -> %2B, @ -> %40, # -> %23, % -> %25
# Боевой прокси от DataImpulse:
PROXY_URL=http://a86f7d1dcf5eb6a46903:46cfc7c210c0cb31@gw.dataimpulse.com:823
```

3. Перезапустите сервисы:

```bash
pm2 restart all
```

## Форматы прокси URL

- **HTTP прокси с авторизацией**: `http://user:pass@proxy.example.com:8080`
- **HTTP прокси без авторизации**: `http://proxy.example.com:8080`
- **HTTPS прокси**: `https://user:pass@proxy.example.com:8080`
- **SOCKS5** (не поддерживается напрямую, нужен HTTP wrapper)

## Проверка работы

После настройки прокси проверьте логи:

```bash
pm2 logs visa-monitor-api | grep "прокси"
```

Должно появиться сообщение: `🔐 Используется прокси для обхода блокировки IP: ...`

Дополнительно можно проверить IP через curl:

```bash
curl -x "http://a86f7d1dcf5eb6a46903:46cfc7c210c0cb31@gw.dataimpulse.com:823" \
  --connect-timeout 10 --max-time 15 \
  "https://api.ipify.org?format=json"
```

## Важные замечания

1. **Безопасность**: Не коммитьте `.env.production` с реальными данными прокси
2. **Производительность**: Прокси может замедлить запросы
3. **Надежность**: Используйте надежные прокси-сервисы
4. **Ротация**: Для больших нагрузок используйте ротацию прокси

## Альтернативные решения

1. **VPN**: Настроить VPN на сервере и использовать его IP
2. **Другой сервер**: Использовать сервер с другим IP адресом
3. **Обращение к администраторам**: Запросить разблокировку IP у администраторов visa API
